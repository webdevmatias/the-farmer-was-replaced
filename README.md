# The Farmer Was Replaced - Algorithms & Automation

Projeto em Python desenvolvido para estudar algoritmos, otimização e organização de código utilizando o jogo The Farmer Was Replaced como ambiente de experimentação.

## Objetivos

- Explorar diferentes algoritmos de navegação.
- Analisar e otimizar movimentação no mapa.
- Desenvolver estratégias para gerenciamento de recursos.
- Aplicar conceitos de Ciência da Computação em um ambiente controlado.
- Melhorar modularização e reutilização de código em Python.

## Estrutura

```text
src/
├── main.py
├── entities.py
├── movements.py
├── utils.py
├── navigation.py
└── ...
```

## Algoritmos Implementados

- Zig-zag traversal
- Grid scanning
- Resource prioritization
- Automated harvesting
- Crop-specific strategies
